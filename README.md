# SalesMonitor
Генератор отчетов с графиками, где из базы данных берутся продажи, составляется разного вида (столбчатая/круговая/линейная) диаграмма. Есть разные расчеты, такие как: средний чек, прирост, доля, итог с начала года. Можно фильтровать по покупателям, по товарам или группировать по определенному периоду. При завершении можно совершить экспорт в Excel. Также можно сделать сравнительный отчет, например, продажи за июль и за октябрь.

<b>Дополнительные возможности:</b>

-Оповещения об упадке продаж или уходе клиентов

-ABC-анализ продаж, XYZ-аналитика спроса

-Фильтры: по маржинальности, по сумме чека, по периодам, сравнение периодов, по дням недели

-Отклонение от среднего для выявления ошибок ввода

-Прогнозирование на основе ML.NET (алгоритм SSA)

-Фоновый мониторинг упадка продаж и рассылка оповещений через SignalR

-Экспорт отчетов в Excel через ClosedXML

<b>Технологии: </b>
ASP.NET Core + Blazor, PostgreSQL, EF Core, MathNet.Numerics, ML.NET, SignalR, ClosedXML
